;テキストボタンのサンプルコード１
;機能だけを分かりやすく解説した単純なサンプルです。

*define
textgosub *lb
game

*lb
textbtnwait %0
if %0>0 goto *textbtn
texec
saveon
return

*textbtn
if %0=2 mesbox "「こんなふうに」を押しました。","テキストボタン"
if %0=4 mesbox "「ヘルプ機能」を押しました。","テキストボタン"
erasetextbtn ;←この命令で、ボタンの色を元に戻します。
print 1
goto *lb


*start
erasetextwindow 0
linkcolor #FF0000,#00FFFF
setwindow 4,324,26,5,20,20,0,0,0,1,1,#2266BB,0,320,639,479
テキストボタンのサンプルです。<2こんなふうに>使います。
<4ヘルプ機能>などを実装するのに便利だと思います。\
end
